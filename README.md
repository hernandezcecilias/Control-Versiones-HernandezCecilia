# Gato Robot - Controles


## Movimiento de las Patas
Las teclas de las patas están programadas para plegar el muslo y la pantorrilla simultáneamente con un solo toque, simulando que el gato reoliega las patas.

| Tecla | Extremidad | Articulaciones que Modifica |
| :---: | :--- | :--- |
| `F`   | Pata Frontal Izquierda | Muslo (1) y Rodilla/Pantorrilla (2) |
| `G`   | Pata Frontal Derecha   | Muslo (3) y Rodilla/Pantorrilla (4) |
| `H`   | Pata Trasera Izquierda | Muslo (5) y Rodilla/Pantorrilla (6) |
| `J`   | Pata Trasera Derecha   | Muslo (7) y Rodilla/Pantorrilla (8) |

---

## Expresiones y Cola Articulada
Controles individuales para darle vida a las partes más expresivas del gato robot.

| Tecla | Parte del Cuerpo | Movimiento |
| :---: | :--- | :--- |
| `K`   | Base de la Cola  | Rota la base principal (9) y el primer segmento (10) |
| `N`   | Medio de la Cola | Rota el segundo segmento de la cola (13) |
| `V`   | Punta de la Cola | Rota el segmento final y el cono (14) |
| `L`   | Oreja Izquierda  | Gira la oreja sobre su propio eje como radar (11) |
| `M`   | Oreja Derecha    | Gira la oreja sobre su propio eje como radar (12) |
| `B`   | Modulo Extra     | Control para la articulación adicional (15) |
